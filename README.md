# NRC Events and Power Webhook
This program serves to automatically fetch:
 -  Reactor powers from the US NRC website   
 (API: `https://www.nrc.gov/reading-rm/doc-collections/event-status/reactor-status/PowerReactorStatusForLast365Days.txt`)
 
 - Events that have occured in nuclear power plants/ nuclear-adjacent locations  
 (parses the page: `https://www.nrc.gov/reading-rm/doc-collections/event-status/event/en.html`)
